# 🎓 Online Learning Platform

A full-stack web application where users can register, login, and access courses. Admin users can manage courses, and students can learn from them.

---

## 🚀 Features

### 👤 Authentication

* User Registration (Admin / Student)
* User Login using JWT Authentication

### 👨‍🏫 Admin Features

* Add new courses
* Delete courses
* Add full course content:

  * Introduction
  * Topics
  * Conclusion

### 🎓 Student Features

* View available courses
* Open course details
* Learn full course content

### 🎨 UI Features

* Beautiful homepage with course preview cards
* Background images for Login & Register pages
* Styled course cards
* Navigation bar

---

## 🛠️ Technologies Used

### Frontend

* React.js
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
